## Advent of Code 2024 solutions
My solutions to [Advent of Code](https://adventofcode.com/2024) problems - 2024 edition.
Solutions are written in C++17 (although most should work with C++14).

### Try it out
To try out any of my solutions with your input, create a file named "input.txt" in the folder for the respective day,
build using your favorite compiler and run it.
